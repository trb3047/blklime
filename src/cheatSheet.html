Git

  • 설정

    유저 정보 설정하기

    $> git config --global user.name "이름"
    $> git config --global user.email "이메일 주소"
    $> git config --global color.ui auto

    설치

    해당 디렉토리에 .git 설치

    $> git init

    Reppo.에서 복사해오기

    $> git clone [git url]

    임시 저장소(STAGE)와 형상(SNAPSHOT) 관리

    현재 Git 상태 확인

    $> git status

    임시저장소(stage)에 로컬 파일 추가

    $> git add [파일명]

    add된 파일을 로컬에 유지하면서 unstage

    $> git reset [파일명]

    staged가 아닌 파일들에서 차이점 보여주기

    $> git diff

    staged 상태에서 아직 commit 되지 않은 파일들의 차이점 보여주기

    $> git diff --staged

    commit 하기

    $> git commit -m "[작업 내용에 대한 메세지]"

  • BRANCH와 MERGE

    브랜치 목록 보기

    $> git branch

    새 브랜치 만들기

    $> git branch [브랜치 이름]

    다른 브랜치로 이동

    $> git checkout [브랜치 이름]

    다른 브랜치에서 가져와 합치기

    $> git merge [브랜치 이름]

    현재 브랜치의 기록을 보기

    $> git log
